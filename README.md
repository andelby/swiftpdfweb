# SwiftPDF Website

Modern, beautiful website for SwiftPDF - a Windows 11 PDF viewer application.

## Files

- `index.html` - Main landing page with features and download links
- `styles.css` - Windows 11 inspired dark theme styles
- `main.js` - JavaScript for interactions and animations
- `privacy.html` - Privacy policy page
- `changelog.html` - Version history and release notes

## Deployment

This website is designed for GitHub Pages deployment.

### Setup

1. Push this repository to GitHub
2. Enable GitHub Pages in repository Settings > Pages
3. Select the `main` branch as the source

The site will be available at: `https://[username].github.io/swiftpdfweb/`

### Adding a Demo Video

Place your demo video file as `swiftpdf-demo.mp4` (and optionally `swiftpdf-demo.webm`) in the root directory. The video player on the homepage will automatically use it.

### Customization

- **Microsoft Store Link**: Update the store link in `index.html` (search for `apps.microsoft.com`)
- **Colors**: Modify CSS variables in `styles.css` to match your brand
- **Features**: Edit the feature cards in `index.html`
