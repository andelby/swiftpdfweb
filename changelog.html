Changelog

All notable changes to SwiftPDF will be documented in this page.

v1.0.0 Initial Release
Added
  • PDF viewing with smooth scrolling and zooming
  • Multi-tab interface for working with multiple PDFs
  • Annotation tools: highlight, ink, free text, stamps
  • Tab drag and drop to reorder or tear off into new windows
  • Windows 11 Mica backdrop support
  • Dark and light theme support
  • Microsoft Office document support (Word, Excel, PowerPoint)
  • Keyboard shortcuts for annotation tools
  • Outline/bookmarks sidebar
  • Thumbnail navigation
  • Search within documents
v1.1.0 Coming Soon
Improved
  • Performance optimizations for large PDF files
  • Better memory management
Added
  • Form filling support
  • Signature annotation tool
  • Recent files quick access
